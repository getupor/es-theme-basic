Chaque fois que je te fournis un nom de système, de consoles, de borne d'arcade ou encore de collections de jeux je souhaite que tu me fournisse une description en anglais et une en Français , n'hésites pas à ajouter des anecdotes si tu en trouves, des exemples de jeux etc.. essaie de formuler ta description d'une manière qui permet de mettre certaines informations entre parenthèses comme le nom de la console, le fabricant, les composants ou encore les dates importantes, n'oublies pas dès le début de la description de donner si nécessaire le nom que porte la machine dans les autres pays comme le japon ou les usa et sois extrêmement bavard car je souhaite de longues descriptions. Chaque description sur une seule ligne, c'est à dire une pour l'anglais et une pour le Français, le tout dans ce format dans un encart code :



		<desc>xxx</desc>

		

		<desc lang="fr">xxx</desc>



Tu dois remplacer "xxx" par les descriptions
